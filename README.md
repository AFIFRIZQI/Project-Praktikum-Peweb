# Project-Praktikum-Peweb
Ini adalah proyek untuk menyelesaikan tugas akhir praktikum pemrograman web.
# Description
Aplikasi CRUD Toko Olahraga

Aplikasi CRUD Toko Olahraga merupakan sebuah aplikasi berbasis web yang digunakan untuk mengelola data produk olahraga. Aplikasi ini dibuat menggunakan bahasa pemrograman PHP sebagai backend, MySQL sebagai basis data, serta HTML, CSS, dan JavaScript sebagai frontend untuk tampilan dan interaksi pengguna.

Aplikasi ini menerapkan konsep CRUD (Create, Read, Update, Delete), yang memungkinkan pengguna untuk:
Menambahkan data produk olahraga baru
Menampilkan daftar produk olahraga
Mengubah (edit) data produk yang sudah ada
Menghapus data produk dari sistem

Struktur aplikasi dipisahkan antara backend dan frontend untuk meningkatkan kerapian kode dan kemudahan pengelolaan. Folder backend berisi file PHP yang menangani logika aplikasi dan koneksi database, sedangkan folder frontend berisi file CSS dan JavaScript yang mengatur tampilan serta interaksi antarmuka pengguna.

Aplikasi ini dirancang dengan tampilan yang sederhana namun modern, dilengkapi dengan validasi data menggunakan JavaScript agar data yang dimasukkan sesuai dan mengurangi kesalahan input. Dengan adanya aplikasi ini, pengelolaan data produk olahraga menjadi lebih efektif, terstruktur, dan mudah digunakan.
